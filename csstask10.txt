2. Create a web layout with a flex container that holds 3 child boxes?

<!DOCTYPE html>
<html>
<head>
  <title>Flexbox Layout</title>
  <style>
    .container {
      /* Make it flex */
      display: flex;
      /* Center horizontally */         
      justify-content: center;  
       /* Center vertically */
      align-items: center; 
       /* Space between boxes */
       gap: 20px;                   
      height: 200px;
      border: 2px solid black;
    }
.box {
      width: 80px;
      height: 80px;
      background-color: blue;
      color: white;
      text-align: center;
      line-height: 80px;
      font-size: 20px;
    }
  </style>
</head>
<body>
<div class="container">
    <div class="box">1</div>
    <div class="box">2</div>
    <div class="box">3</div>
  </div>
</body>
</html>
