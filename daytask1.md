**1. What is HTML?**

Html stands for Hypertext markup language.html is a structure of webpages.

History:

\*HTML was created by tim berners lee in 1991.Current version is html5.

**Why HTML is Used?**

\*It build a websites and web application.

\*it is used to create a display text, image and videos



**2.Understand <!DOCTYPE>, <html>, <head>, <body>.Create a simple html page?**

<!DOCTYPE HTML> - Define the html version

<html> - root element of page

<head> - contains page information

<body> - contains a visible content

**Example code:**

<!DOCTYPE html>

<html>

<head>

<title>SLA</title>

</head>

<body>

<h1>Welcome to SLA Institute</h1>

<p>This is my first task</p>

<p>I am learning HTML in Day1</p>

</body>

</html>
