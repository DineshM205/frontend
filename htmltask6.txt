1.Learn the purpose of <form> in HTML. Create a basic form structure with action and method.

Purpose of <form> in HTML

It is used to collect user input and send that data to a server for processing.

KEY ATTRIBUTES OF <form>
1.action
specifies where the form data is sent
2.method
Defines how data is sent
Two common types:
GET (data visible in URL)
POST(data hidden)

EXAMPLE:
<!DOCTYPE html>
<html>
<head>
    **<title>Simple Form</title>**

</head>
<body>
<h2>Student Form</h2>
<form action="submit.php" method="POST">
<!-- Name -->
<label for="name">Name:</label>
<input type="text" id="name" placeholder="Enter username"><br><br>
<!-- Email -->
<label for="email">Email:</label>
<input type="email" id="email" placeholder="Enter email"><br><br>
<!-- Password -->
<label for="password">Password:</label>
<input type="password" id="password"><br><br>
<!-- Submit Button -->
        **<button type="submit">Submit</button>**

</form>
</body>
</html>

4.Learn <label> for accessibility and placeholder for hints. Connect labels with inputs using for and id.

<label> definition:
It is used to define a description for an input field in a form.

PLACEHOLDER(HINT):
Shows a hint inside the input box
Disappears when user starts typing

EXAMPLE:
<label for="name">Name:</label>
<input type="text" id="name" placeholder="Enter your name">

5.Use required to make fields mandatory. Practice building a signup form where name, email, and password are required.
REQUIRED DEFINTION:
required is an attribute used to make a field mandatory.
User cannot submit the form without filling it.

EXAMPLE:
