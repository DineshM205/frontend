1.Learn what semantic tags are and why they are important for SEO & accessibility. Compare semantic vs non-semantic tags.

*What are Semantic Tags?
Semantic tags are HTML tags that clearly describe the meaning and purpose of the content inside them.
They help both browsers and developers understand the structure of a webpage.

Examples of semantic tags:
<header> – defines the top section of a page
<nav> – contains navigation links
<section> – represents a section of content
<article> – independent content like a blog post or news article
<aside> – side content (like ads or sidebars)
<footer> – bottom section of a page
*Why Semantic Tags are Important
1️⃣ SEO (Search Engine Optimization)
Search engines understand the structure and importance of content.
Example:
<article> → search engine knows it is main content
<nav> → navigation links
<header> → page heading
This helps search engines index the website better and improve ranking.
2️⃣ Accessibility
Semantic tags help screen readers used by visually impaired users.
Example:
Screen readers know when navigation starts (<nav>)
They understand main content (<article>)
So the website becomes more accessible and user-friendly.
3️⃣ Better Code Structure
Semantic HTML makes code:
Easy to read
Easy to maintain
Easy for developers to understand

2.Practice creating a webpage with a header and footer. Add title, logo, and footer info.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Website</title>
</head>
<body>

    <!-- Header Section -->
    <header>
        <img src="pic1.png" alt="Website Logo" width="100">
        <h1>My Website</h1>
    </header>
<hr>

   <!-- Main Content -->
    <section>
        <h2>Welcome</h2>
        <p>This is my practice webpage with a header and footer.</p>
    </section>

    <hr>

    <!-- Footer Section -->
    <footer>
        <p>© 2026 My Website</p>
        <p>Contact: myemail@example.com</p>
    </footer>

</body>

</html>
