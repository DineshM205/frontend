1.Create a box that takes up 80% of the screen width but no more than 600px.
2.Set a paragraph’s font size to 1.5rem and explain how it behaves on different devices.
3.Use vw to make a heading that scales with screen size.
4.Write a media query that changes background color below 768px.

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        #s1 {
            border: 2px solid black;
            width: 80%;
            max-width: 600px;
            height: 100px;
            font-size: 1.5rem;
            text-align: center;
            background-color: white;
            color: red;
          }
    @media(max-width:768px) {
            #s1 {
                background-color: blueviolet;
            }
        }
       header {
            font-size: 5vw;
        }
    </style>
</head>
<body>
    <header>Response webpage</header><br>

    </div>
