1.Style an HTML snippet using different CSS selectors?
<html>
<head>
    <style>
        /* Element Selector */
        h1 {
            color: blue;
        }

       /*Class Selector */
        .para {
            color: green;
            font-size: 18px;
        }

        /* ID Selector */
        #main-title {
            text-align: center;
            background-color: lightgray;
        }

        /* Group Selector */
        h2, p {
            font-family: Arial;
        }

        /* Universal Selector */
        * {
            margin: 10px;
        }

        /* Descendant Selector */
        div p {
            color: red;
        }

        /* Hover Selector */
        button:hover {
            background-color: yellow;
        }
    </style>
</head>
<body>
<h1 id="main-title">Welcome</h1>
<h2>Sub Heading</h2>
<p class="para">This is a paragraph using class selector.</p>
<div>
<p>This paragraph is inside div (descendant selector).</p>
</div>
<button>Click Me</button>
</body>
</html>
