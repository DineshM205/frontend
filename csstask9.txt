 1. Create a div with a class .box.
<!DOCTYPE html>
<html>
<head>
    <title>Create Box</title>
    <style>
        .box {
            width: 200px;
            height: 100px;
            background-color: lightgray;
}
    </style>
</head>
<body>
    <div class="box"></div>
</body>
</html>

 2. Inside the .box, place some text content.
<!DOCTYPE html>
<html>
<head>
    <title>Box Content</title>
    <style>
        .box {
            width: 200px;
            height: 100px;
            background-color: lightgray;
        }
    </style>
</head>
<body>
    <div class="box">
        This is the content inside the box.
    </div>
</body>
</html>

3. Style the .box to clearly show:
<!DOCTYPE html>
<html>
<head>
    <title>Styled Box</title>
    <style>

            width: 250px;
            background-color: #f0f0f0;
        }
    </style>
</head>
<body>
    <div class="box">Styled box</div>
</body>
</html>

 5. border:a visible outline.

6. padding: space inside the border, around the content.
<!DOCTYPE html>

<body>
