 1.Learn what attributes are in HTML. Explore how they provide extra information about elements (e.g.,href,src,alt).

what is the attributes:

2.Learn how to use class for styling multiple elements with CSS. Practice applying same class to multiple tags.

A class is an attribute used to apply the same CSS style to multiple HTML elements.A class is an attribute used to apply the same CSS style to multiple HTML elements.

EXAMPLE:
<style>
.title{
    color: blue;
    text-align: center;
}</style>
<h1 class="title">Welcome</h1>
<p class="title">This is my website</p>
<div class="title">Learning HTML and CSS</div>

3.Understand the uniqueness of id. Use it for styling and JavaScript targeting. Practice assigning unique IDs.

UNIQUENESS OF ID:
The id attribute is used to give a unique identifier to a single HTML element.

Each id must be used only once in a webpage.
It helps CSS styling and JavaScript targeting.

EXAMPLE:
<!DOCTYPE html>
<html>
<head>
<style>
#header{
   color: blue;
   text-align: center;
}
#para{
   color: green;
}
#footer{
   background-color: lightgray;
}
</style>
</head>
<body>
<h1 id="header">Welcome to My Website</h1>
<p id="para">This is my first HTML page.</p>
<div id="footer">Copyright 2026</div>
</body>
</html>

src means source.
the browser where the image file is located.

9.Create a webpage with navigation links, multiple sections using IDs, and images with proper alt text and titles.
