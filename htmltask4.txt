1. Learn difference between <b> (bold styling) and <strong> (semantic importance). Practice both in sentences.

Ans:	<b> → visual bold styling only. It used no special importance.
	<strong> → semantic importance (important text).Has meaning for SEO and screen readers

Example meaning:

<b> → styling
<strong> → screen readers treat it as important

2. Understand <i> (italic styling) vs <em> (emphasis). Use them in a paragraph.

Ans:	<i> → italic style only
	<em> → emphasized text (semantic meaning)

	<p>
	The movie <i>Interstellar</i> is a science fiction film.
	But I <em>really</em> love its soundtrack.
	</p>
Here:
	<i> is for titles or style
	<em> adds emphasis

3.Practice underlining text with <u>, highlighting with <mark>, and making side notes with <small>

Ans:	<u> → underline
	<mark> → highlight text
	<small> → side notes or smaller text

	<p>
	Please read the <u>instructions carefully</u>.
	The deadline is <mark>tomorrow</mark>.
	<small>(Late submissions may not be accepted)</small>
	</p>

4.Create hyperlinks using <a>. Add external links, internal links, and “back to top” links.
Ans: External link
	<a href="https://www.google.com">Visit Google</a>

Ans: Internal link
	<a href="about.html">Go to About Page</a>

Ans: Back to top link
	<a href="#top">Back to Top</a>

Example with page anchor:

<h1 id="top">My Website</h1>

<p>Some long content here...</p>

<a href="#top">Back to Top</a>

5.Learn how <button> works inside forms and as clickable UI elements. Practice adding button text and attributes.
Ans: Button inside a form
	<form>
	<button type="submit">Submit Form</button>
	</form>
Ans: Button as UI element
	<button onclick="alert('Button clicked!')">
	Click Me
	</button>

Attributes:

	type="submit"

	type="button"

	type="reset"
